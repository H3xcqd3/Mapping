# Roadmap

| Milestone | Outcome | Exit criteria |
|---|---|---|
| v0.1 Toolchain | Safe repository and automation | Scripts parse, safe dry-runs pass, local paths remain configurable |
| v0.2 Terrain | First traversable terrain | Client and FDS load it; collision and boundaries pass |
| v0.3 Bases | Expanded terrain and functional team bases | 450 by 320 scale retest, four or five buildings per team, spawns, terminals, and vehicle clearances pass |
| v0.4 Harvesters | Working economy loop | Both harvesters complete repeatable routes |
| v0.5 Playable | End-to-end test map | Core test plan passes and balance issues are documented |
| v1.0 Public beta | Community-ready package | Clean package, documentation, compatibility, and release notes |

Long-term direction: evolve the reusable tooling and asset library into the OnOeS AI Mapping Suite. Maps remain concrete proving grounds; the suite must not generate or change gameplay design without designer approval.
