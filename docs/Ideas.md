# Ideas

Ideas are not approved work. Each item requires a design decision before implementation.

- Multi-map workspace with `Maps`, `Shared`, and `Tools` areas.
- AI-assisted terrain concepts, height maps, route suggestions, and texture palettes.
- Validator adapters for LevelEdit exports once their inputs are documented.
- Asset catalogue with thumbnails, source ownership, dependencies, and map usage.
- Automated checks for world bounds, floating objects, broken links, and waypaths when verified machine-readable data becomes available.
