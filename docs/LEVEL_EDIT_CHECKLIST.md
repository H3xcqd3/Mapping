# LevelEdit checklist

## World

- [ ] Terrain is visible and uses the intended textures.
- [ ] Collision works on all playable surfaces.
- [ ] There are no terrain holes or unintentionally inaccessible areas.
- [ ] World boundaries are complete.
- [ ] Infantry and vehicle blockers are intentional and tested.
- [ ] Damage zones are correctly bounded.
- [ ] Vis sectors are configured where applicable.
- [ ] Lighting is consistent and playable.

## Teams and buildings

- [ ] GDI and Nod spawn points work and cannot spawn players inside geometry.
- [ ] All building controllers are linked correctly.
- [ ] Purchase terminals work for the correct team.
- [ ] Refineries and harvesters are configured.
- [ ] Harvester waypaths complete a full cycle.
- [ ] Vehicle factory exits have sufficient clearance.
- [ ] Base defences match the approved design.

## Release readiness

- [ ] Texture paths resolve with supported extensions.
- [ ] The map filename exactly matches the configured map name.
- [ ] Team layout has received a balance review.
- [ ] Local client test passes.
- [ ] Local dedicated-server test passes.
